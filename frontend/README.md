# DoctorOnDoor - Professional Healthcare Services Website

A modern, responsive healthcare services website built with React, TypeScript, and Vite. Features comprehensive home healthcare services including nursing, elderly care, physiotherapy, and laboratory testing.

## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom gradients
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Routing**: React Router
- **Backend**: Express.js with MongoDB

## Features

- Modern responsive design with full-screen layouts
- Professional healthcare service pages
- Smooth animations and transitions
- Mobile-first responsive design
- Contact forms and booking system
- Service information and pricing

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Services

The website includes the following healthcare services:

3. **Physiotherapy** - Physical therapy and rehabilitation
4. **Health Checkups** - Preventive health screenings
